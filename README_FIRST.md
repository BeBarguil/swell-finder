## 🏄‍♂️ Sobre o App

**Surf Spot Finder** ajuda surfistas a encontrar praias próximas com boas condições de surf baseado em:

- 📍 Geolocalização em tempo real
- 🌊 Informações de swell (altura das ondas)
- 💨 Dados de vento
- 🎯 Nível de dificuldade (iniciante, intermediário, avançado)
- 📊 Histórico de condições
