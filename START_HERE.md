## 🚀 3. FAZER UPLOAD PARA O GITHUB

**Abra o Terminal:**
- **Mac:** Cmd + Espaço → "Terminal"
- **Windows:** Git Bash

**Execute estes comandos:**

```bash
# Vá para o Desktop
cd ~/Desktop

# Entre na pasta
cd surf-spot-finder

# Verifique que está no lugar certo (deve mostrar README.md, Gemfile, etc)
ls

# Execute o upload
git init
git add .
git commit -m "feat: initial commit - Surf Spot Finder app"
git branch -M main
git remote add origin https://github.com/BeBarguil/swell-finder.git
git push -u origin main
```

**Quando pedir senha:**
1. Crie um token: https://github.com/settings/tokens/new
2. Marque: ☑️ `repo`
3. Copie o token
4. Use como senha
